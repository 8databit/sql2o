Maven repository

As of version 1.3.0 sql2o is hosted on maven central. To add sql2o as a dependency in your maven project, put the following in your pom.xml

{% highlight xml %} org.sql2o sql2o {{ site.data.config.latestversion }} {% endhighlight %}

 

Versions prior to 1.3.0 are hosted on the old repository on github.
